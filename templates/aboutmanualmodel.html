About Manual Model


How The Model Works


Under this mode, a worker will watch the video of the task. And give a score base on how the participant finish the task.
In our demo, we designed a form to let worker input the scores.
Our model use this data to train and test the model performance. In the evaluate phase, the model achieved 68% accuracy.
The prediction is not an medical diagnosis, it just remind user to pay attention to health status.


The Algorithm


We use XGBoost model to implement the machine learning model.
XGBoost is a decision-tree-based ensemble Machine Learning algorithm that uses a gradient boosting framework.
In prediction problems involving unstructured data (images, text, etc.), artificial Neural Networks tend to
outperform all other algorithms. However, when it comes to small-to-medium data, decision tree based algorithms
are considered best-in-class right now.
