About Sensor Data Model


The Data Set
Our Sensor Data Model used the data set from CASAS (Center for Advanced Studies in Adaptive Systems) in Washington State University.
The project provided a data set including 400 pieces of data, each contained the record from the sensors generated from their experiment.
Our model use this data to train and test the model performance. In the testing phase, our model achieved an total 66% accuracy.

The Algorithm
We use XGBoost model to implement the machine learning model.
XGBoost is a decision-tree-based ensemble Machine Learning algorithm that uses a gradient boosting framework.
In prediction problems involving unstructured data (images, text, etc.), artificial Neural Networks tend to
outperform all other algorithms. However, when it comes to small-to-medium data, decision tree based algorithms
are considered best-in-class right now.
